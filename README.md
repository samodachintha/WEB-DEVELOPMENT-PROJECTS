# Surprise Webpage

This project is a simple webpage created to surprise my girlfriend. It showcases a beautiful layout with heartfelt messages and images.

## Project Structure

```
surprise-webpage
├── css
│   └── styles.css
├── index.html
└── README.md
```

## Files Description

- **index.html**: The main HTML document that contains the structure of the webpage, including headings, paragraphs, images, and links.
- **css/styles.css**: The stylesheet that defines the layout, colors, fonts, and other visual aspects to enhance the appearance of the HTML content.

## How to View the Webpage

1. Clone or download the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Enjoy the surprise!
